# brain-tumore-detection-using-cnn
Brain Tumor Detection using Convolutional Neural Networks (CNN) is a deep learning project designed to accurately identify and classify brain tumors from MRI scan images. This system uses image preprocessing, feature extraction, and CNN-based classification techniques to detect abnormalities in brain scans, helping improve early diagnosis and medical decision-making. The project aims to provide an efficient, automated, and reliable solution for brain tumor detection with high accuracy, reducing manual effort and supporting healthcare professionals.
