# рџ§  Project Overview

Brain Tumor Detection Using CNN is a deep learning-based medical image classification project designed to identify the presence of brain tumors from MRI scan images.

The project utilizes Convolutional Neural Networks (CNNs) to automatically extract image features and classify MRI images into tumor and non-tumor categories.

This project demonstrates the application of Artificial Intelligence, Computer Vision, and Deep Learning in healthcare diagnostics.
